## 📖 Sobre o Projeto

O **Global Conflict Monitor** é uma plataforma desenvolvida para analisar conflitos armados ao redor do mundo utilizando conceitos de:

* Modelagem Relacional
* Normalização de Dados
* Consultas SQL
* Visualização Geográfica
* Análise Histórica e Geopolítica

Mais do que um projeto acadêmico, o objetivo foi transformar dados históricos em uma experiência visual capaz de demonstrar os impactos humanos, culturais e econômicos causados pelos conflitos ao longo do tempo.

---

## 🎯 Objetivos

* Organizar dados históricos de conflitos armados.
* Aplicar conceitos de Banco de Dados em um cenário real.
* Construir consultas analíticas utilizando SQL.
* Criar visualizações interativas para facilitar interpretações.
* Evidenciar os impactos globais da violência organizada.

---

## 🗂 Dataset

Os dados utilizados foram obtidos através do:

**UCDP — Uppsala Conflict Data Program**

Uma das maiores bases acadêmicas do mundo sobre conflitos armados e violência organizada.

🔗 https://ucdp.uu.se/downloads/

O dataset contém informações sobre:

* Guerras civis
* Conflitos territoriais
* Violência unilateral
* Confrontos entre Estados
* Fatalidades registradas
* Localização geográfica dos eventos

## 🧠 Estrutura do Banco de Dados

O projeto passou por um processo de normalização para eliminar redundâncias e aumentar a integridade dos dados.

### Benefícios obtidos

✅ Redução de duplicidade

✅ Melhor organização

✅ Maior desempenho em consultas

✅ Integridade referencial através de chaves primárias e estrangeiras

---

## 🔍 Consultas SQL Desenvolvidas

O projeto inclui análises como:

### 📈 Conflitos por Ano

Identificação da evolução histórica dos conflitos.

### ☠️ Fatalidades por Período

Análise temporal do impacto humano.

### 🤝 União de Atores

Relacionamento entre grupos, países e organizações envolvidas.

### 🌎 Intensidade por Região

Comparação geográfica da violência registrada.

## 🗺️ Mapa Interativo

## 🌐 Impacto Social

Além dos números, o projeto busca evidenciar como os conflitos afetam sociedades inteiras.

### Consequências observadas

* Migração forçada
* Destruição de infraestrutura
* Instabilidade econômica
* Fragilidade política
* Traumas sociais
* Aumento da pobreza

---

## 🏛️ Impacto Cultural

Conflitos não causam apenas perdas humanas.

Eles também provocam:

* Destruição de patrimônios históricos
* Desaparecimento de tradições
* Perda de memória cultural
* Deslocamento de comunidades
* Enfraquecimento da diversidade cultural
